☁️ Paso a paso: Creación de Cuenta en la Nube Pública (Microsoft Azure con Correo Institucional)

Este documento explica paso a paso cómo se creó una cuenta en Microsoft Azure usando una cuenta de correo institucional/universitaria, lo cual permite acceder al beneficio Azure for Students, sin necesidad de tarjeta bancaria y con créditos gratuitos.

Paso 1: Ingreso al sitio oficial de Azure

  1. Abrir el navegador.
  2. Ir al sitio: https://azure.microsoft.com
  3. Hacer clic en el botón “Para estudiantes” o “Comienza gratis”.

Paso 2: Iniciar sesión con correo institucional

  1. En la página de registro, iniciar sesión con una cuenta institucional (por ejemplo, nombre@universidad.edu).
  2. Si no existía previamente una cuenta Microsoft con ese correo, se crea una durante este paso.
  3. El sistema reconoce el dominio educativo y habilita automáticamente el plan para estudiantes.

Paso 3: Verificación por teléfono

  1. Ingresar un número de teléfono válido.
  2. Elegir método de validación (mensaje de texto o llamada).
  3. Ingresar el código recibido para validar identidad.

Paso 4: Activación del paquete Azure for Students

Una vez validado el correo institucional y el teléfono:

  • Se activa el plan Azure for Students.
  • Beneficios incluidos:
    • $100 USD en créditos gratuitos
    • Acceso a servicios premium como:
      • Máquinas virtuales
      • Bases de datos SQL
      • Web apps
      • Azure Functions
      • Almacenamiento (Blob)
    • Vigencia del crédito: 12 meses o hasta agotarse.

Paso 6: Acceso al portal de Azure

  1. Ingresar a: https://portal.azure.com
  2. Acceder con el correo institucional y comenzar a usar los recursos.
  3. El usuario ya puede crear:
    • Aplicaciones web
    • Bases de datos
    • Servicios de backend
    • Redes virtuales, entre otros.

Paso 7: Seguridad y autenticación

  1. Habilitar la autenticación multifactor (MFA) desde Azure Active Directory.
  2. Opcional: establecer alertas de uso para evitar consumir los créditos por accidente.

✅ Resultados finales

✔️ Cuenta activa
✔️ Plan educativo habilitado
✔️ Portal operativo
✔️ Créditos listos para usar
